procedure

(draw d ...) → pict?

  d : drawable?
Converts each argument to a pict. The picts are then superimposed with cc-superimpose. The bottom layer is from the first argument, and the top layer from the last argument.

If there are no argument, then (draw) returns a blank pict of width (curve-pict-width) and height (curve-pict-width).

The objects draw can render to picts are: curve, pict, pt, bez, and, label.

(filldraw c) → pict?

  c : curve?
Draws, then fills, the curve c.

That is, filldraw is equivalent to (draw (fill c)).
